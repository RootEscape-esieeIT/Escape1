# PREFIXE :
Add : créer un nouveau fichier/dossier/fonctionnalité (pour ce dernier on met le nom du bloc de code)
Update : modifier un fichier
Delete : supprimer un fichier

# PATH :
Chemin vers le fichier conserné (le plus concerné si plusieurs fichiers pris en comptes)

# séparation :
une virgule sépare plusieurs fichiers, (si plusieurs sont Add/delete/update)
un point virgule sépare plusieurs catégorie

exemple :

Add site.html

Update site.html, site.css; Add settings.json
